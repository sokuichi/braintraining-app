            Method notes

            Training uses known creativity practice patterns.

            • Divergent thinkingQuantity, flexibility, and elaboration practice are common targets in effective creativity training.
            • IncubationBreaks work best after preparation, so the app uses preparation before rest-based prompts.
            • Analogical transferTransfer drills push structure mapping from a source domain to a target problem.
            • Deliberate practiceEvery module gives goals, feedback, and difficulty changes based on performance.

            Research base

            The app objectifies training by turning each method into a task with a target, a score, feedback, and adaptive difficulty.

            Deliberate practice

            Training should use clear goals, focused reps, immediate feedback, and adjustments at the edge of ability.

            Ericsson et al. 1993

            Retrieval practice

            The Question Gym uses active recall and generated tests because retrieval practice can improve long-term retention.

            Roediger and Karpicke 2006

            Distributed practice

            The progression system rewards repeated sessions instead of one long grind because spacing improves retention across many recall tasks.

            Cepeda et al. 2006

            Desirable difficulty

            Difficulty ramps are meant to stay challenging but useful: hard enough to force effort, not so hard that practice collapses.

            Interleaving

            Math interleaving

            Question Gym mixes problem families so you practice selecting a solution strategy instead of repeating a memorized procedure.

            Rohrer and Taylor 2007

            Feedback after tests

            Answer feedback is shown after each generated item to strengthen correct retrieval and reduce repeated lure errors.

            Butler and Roediger 2008

            Retrieval over review

            Question Gym and Systems Lab favor answered problems over rereading because retrieval practice can outperform elaborative review on delayed tests.

            Karpicke and Blunt 2011

            Inductive interleaving

            Mixed categories and problem families support discrimination learning: deciding what kind of problem this is becomes part of the task.

            Kornell and Bjork 2008

            Feedback model

            Explanations are designed to answer where you are going, how you are going, and what to adjust next.

            Hattie and Timperley 2007

            Transfer limits

            The app separates trained-task mastery from broader claims because cognitive-training transfer evidence is mixed and often narrow.

            Sala and Gobet 2019

            Cognitive load

            Creativity training

            Incubation

            The app separates preparation from evaluation because incubation effects are strongest after the problem has been loaded.

            FAQ

            Quick answers for training, evidence, data, and terminal control.

            Is this a real training app now?

            Yes. Progress is stored per module, difficulty adapts from recent objective scores, and each task has a scored output: accuracy, speed, recall matches, idea structure, calibration error, or systems accuracy.

            What data is saved?

            The app saves XP, sessions, streak, settings, module difficulties, recent scores, and a small session history in this browser's localStorage. It does not create a server account or send your data anywhere.

            Can I back up or move progress?

            Use backup to copy JSON, download backup to save a restorable file, or Select backup file to load a JSON backup into the import box. Press Import only after the file is loaded.

            Does n-back increase IQ?

            The app does not claim broad IQ transfer. N-back is treated as working-memory updating practice, and the app reports task progress directly. The Research page includes transfer caveats because broad cognitive-training transfer is mixed.

            Why are tasks adaptive?

            Adaptive difficulty tries to keep practice near the edge of ability. Strong recent scores raise stored difficulty; weak recent scores lower it so practice can rebuild instead of becoming random failure.

            What is scientifically backed here?

            The app leans on retrieval practice, spacing, interleaving, feedback, deliberate practice, calibration, spatial training, and creativity-training research. The methods are mapped to measurable drills instead of vague motivational claims.

            What changed for relational reasoning?

            RRT Lab trains researched relation families directly: same/opposite, more-less, before-after, class inclusion, transitive inference, analogical mapping, proportional relations, and matrix-style rule induction.

            How does imported data work?

            Paste or select CSV/JSON exports in Terminal Data Port. The app stores only sanitized local rows, summarizes them by module, and can export those imported rows separately with export imports.

            How do I make it harder?

            Use preset brutal, hard, challenge systems, set all 10, set depth extreme, set load heavy, or vfx max. Individual modules can be tuned with set memory 9 or the Terminal customizer.

            How do I make it safer or calmer?

            Use preset calm, set motion reduced, set sfx off, vfx off, set load short, or set adaptive off. The wipe confirm command clears this app's local progress key after you have made a backup.

            What does the terminal actually control?

            It can open and start modules, set all difficulties, cap difficulty, randomize or normalize modules, switch n-back modes, change theme, volume, density, VFX, depth, session load, adaptive behavior, and export or import local data.

            What should I run first?

            Try daily plan, then start memory, start questions, start focus, and start spatial. Use stats afterward to see whether the app raised or held your difficulty.
