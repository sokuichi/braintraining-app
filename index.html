        Method notes

        Training uses known creativity practice patterns.

        • Divergent thinkingQuantity, flexibility, and elaboration practice are common targets in effective creativity training.
        • IncubationBreaks work best after preparation, so the app uses preparation before rest-based prompts.
        • Analogical transferTransfer drills push structure mapping from a source domain to a target problem.
        • Deliberate practiceEvery module gives goals, feedback, and difficulty changes based on performance.

        Research base

        The app objectifies training by turning each method into a task with a target, a score, feedback, and adaptive difficulty.

        Deliberate practice

        Training should use clear goals, focused reps, immediate feedback, and adjustments at the edge of ability.

        Ericsson et al. 1993

        Retrieval practice

        The Question Gym uses active recall and generated tests because retrieval practice can improve long-term retention.

        Roediger and Karpicke 2006

        Creativity training

        Creative Sprint scores fluency, flexibility, and elaboration because these are common targets in effective creativity training.

        Scott, Leritz, and Mumford 2004

        Incubation

        The app separates preparation from evaluation because incubation effects are strongest after the problem has been loaded.

        Sio and Ormerod 2009

        Working memory

        Memory Lab uses n-back style load to train updating and monitoring. The app treats transfer claims cautiously and scores the task itself.

        Jaeggi et al. 2008

        Attention control

        Focus Switch uses conflict between word and ink color to train inhibition, task focus, and fast response control.

        Stroop 1935

        Objective scoring model

        Each module has observable outputs, not generic motivation points.

        • AimHits, misses, accuracy, goal completion, best streak.
        • QuestionsCorrect answers across generated problem types.
        • MemoryExact n-back symbol and position decisions.
        • FocusInk-color accuracy and average reaction time.
        • CreativityIdea count, variety, elaboration, and constraint satisfaction.
        • TransferSource structure, target mapping, prototype rule, and test quality.
